The *CCGR* is a collection of card game rules, all playable with a standard deck of cards. The games strive to adhere to the simplest possible ruleset while providing meaningful gameplay with both luck and skill in appropiate measures. All of this presented in a gloriously boring, fast-loading, well-behaved website. **Games with a version below 1.0.0 are considered experimental and requires testing.**

*CCGR* is intended to be the definitive source of included rules and will change over time as collaborators provide feedback, new games are included and older rulesets are updated. In other words, the ultimate goal is to provide library of games with zero faff. To that end, **everyone** is encouraged to participate, both with changes to current rules but also with completely new games.

Another important value of this project is to provide *meaning*. A lot of game rules are written as indisputable, comsic truths. The creator of this project strives to provide the reasoning behind why rules exist (and hopefully start a trend).

*CCGR* aspires to evolve greatly over time. You are invited to participate: https://github.com/superDuperCyberTechno/ccgr
