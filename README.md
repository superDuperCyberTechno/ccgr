# Contemporary Card Games Repository
In essence, the *CCGR* is a collection of rules for card games. It is also a community of people who collaborate to devise new games. *CCGR* is not a library of card games you'd normally find in books or learn from your grandma. - Most of the rules are created specifically to be hosted here, others have been graciously added from other sources. Together, they share a common theme; creative urges to make great card games.

The role of *CCGR* is to...
- Consistent layout: Easy-to-understand text within a rigid template.
- Elegant rules: You should be able to understand the mechanics within 1 or 2 games, that means few moving parts and a solid rules base.
- Reviewable: None of the games are ever considered 100% done. Small changes are embraced, either to improve rules readability or by tweaking the rules themselves. This is a feature, not a bug.
- Fun: The are all tested and improved upon for maximum yielded entertainment.
